CONSEPTO GIT

Git es un software que nos permiten que nos permiten crear respaldo del proyecto en un repositorio de control completo estos repositorios periten trabajar sin conexion o de forma remota con facilidad, Git es un sistema de control de versiones distribuido, lo que significa que un clon local del proyecto es un repositorio de control de versiones completo. Estos repositorios locales plenamente funcionales permiten trabajar sin conexión o de forma remota con facilidad.

Principales Comandos Git

  • GIT INIT: Este comando sirve para iniciar git.
  • GIT ADD: Este comando permite agregar area de reparacion.
  • GIT CONFIG: Permite al usuario identificar el usuario el nombre y correo.
  • GIT COMMIT: Permite realizar un comentarios sobre los cambios realizados.
  • GIT STATUS: Muestra el estado del directorio de trabajo y del area de entorno del ensayo.
  • GIT LOG: Permite explorar el historial de repositorio.
  • GIT REMOTE: Permite crear, ver y eliminar conexiones con otros repositorios.
  • GIT BRANCH: Te permite crear, enumerar y eliminar ramas, asi como cambiar su nombre.
  • GIT CHECKOUT: Permite desplazarte entre las ramas creadas.
  • GIT PUSH: Guarda todo en la nube de git.
